Welcome to a WebxPST-compatible page. This page is a demo, please refer to the source to learn how to make a nicer one for your real tutor deployment.
To start:
  1. Install the WebxPST extension if it isn't already installed, from here
  2. Open the WebxPST sidebar (if it is not already open)
  3. Choose a task from the sidebar menu and press start
  4. Browse to the web application and begin the task (for example, Web Authoring System)
