Welcome to Instruction on Applying for Employment

You're about to apply for employment with Acme, Inc. While you're doing so, an xPST tutor will be available to guide you if needed. You can ask for a hint on what to do at any time.

Note: If you're not using Firefox 3, please switch to it.

To start:

  1. Install the WebxPST Firefox Add-on if it isn't already installed, by clicking here.
  2. Open the WebxPST sidebar (if it is not already open) by choosing the Tools menu -> WebxPST Tutor.
  3. Browse to the Employment Application.
